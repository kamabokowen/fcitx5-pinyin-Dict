# Fcitx 5拼音输入法的字典/词库

My self-created pinyin dictionary for Fcitx 5.

部分词库由搜狗细胞词库转化而成。
适用于Linux/Android的 Fcitx 5输入法（小企鹅输入法5）

文件夹内txt文件为libpinyin文件，您可以直接借此预览包含的词语。

dict文件为Fcitx 5使用的词库文件，您可以直接导入到您的输入法中。
